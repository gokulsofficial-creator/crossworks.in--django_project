# CrossWorks.in (Django Project)

This is a Django web application of the company where multiple users can submit info data and an admin can view all submissions using the Django Admin panel.

## Features
- Form submission
- Database storage
- Admin panel access
- HTML & CSS frontend

## Technology Used
- Python
- Django
- HTML
- CSS

## How to Run
1. Install Python
2. Install Django
3. Run:
   python manage.py runserver
